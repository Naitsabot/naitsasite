# Chargepoint Placement finder (Fall 2023)

This is the first (well second, but thats besides the point) semester project of a total of six in my software bachelor. A requirement from the study regulations for the was to write the project in C. As it is the first project the focus is on basic imperative practices.

This project takes focuses on the placement of chargers for electric vehicles. This is to be done by developing a program that can map out the ideal placements of the chargers.

## Abstract

The continued growth and adoption of electric vehicles (EV) comes with an increase in the demand for EV charging stations.
